# Neo Pelican Theme (Aether)

A highly interactive, modern, neo-brutalist / glassmorphism-inspired theme for Pelican. 

## Features
- **Glassmorphism Header**: Beautiful blurred navigation bar.
- **Cyberpunk Accents**: Subtle glowing gradients using CSS variables.
- **Micro-Animations**: Smooth hover effects on cards and scroll-based entry animations via IntersectionObserver.
- **Typography**: Uses modern *Outfit* and *JetBrains Mono* from Google Fonts.
- **Fully Responsive**: Built with modern CSS Grid and Flexbox.

## Installation
1. Put this `neo-pelican` folder in your Pelican themes directory.
2. Update your `pelicanconf.py`:
   ```python
   THEME = 'pelican-themes/neo-pelican'
   ```

Enjoy your fresh new blog!
